# コードレビュー結果（2026-03-13）

対象: `8312a73 install icon on fullscreen mode`

## 指摘事項

1. **[High] アプリタイトルのバージョンが意図せず巻き戻っています**
   - `index.html` の `<title>` が `v50.23` から `v50.0` に変更されています。
   - 直近マージ履歴でも `v50.23` へ整合させたコミットがあるため、回帰の可能性が高いです。

2. **[Medium] CSSコメントに `//` が使われており無効です**
   - `<style>` 内の `canvas` ルールで `// max-width`, `// max-height` が使われています。
   - CSSで有効なのは `/* ... */` のみのため、構文エラー扱いになり保守性を落とします。

3. **[Medium] フルスクリーン右クリック時の対象レールがカーソル位置と一致しない可能性があります**
   - `contextmenu` イベントの `isFullscreen` 分岐で `selectedShape` をそのまま `fsMenuTargetRail` に転用しています。
   - クリック座標からのヒットテストを行っていないため、直前選択の別レールにメニューが出る可能性があります。

## 推奨対応

- `<title>` を現在の配布バージョンに戻す（少なくとも `v50.23` と整合）。
- CSSコメントを `/* ... */` へ修正する。
- フルスクリーン分岐でも `showContextMenu` と同等のヒットテストを実施して `fsMenuTargetRail` を決定する。
